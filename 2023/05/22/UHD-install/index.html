发表于2023-05-22|更新于2023-05-22
|阅读量:

​ USRP UHD 是 USRP Universal Hardware Driver - USRP 硬件通用驱动的缩写。它的目的是为 USRP 提供通用硬件驱动。USRP UHD 的目的是为 USRP 目前和未来的产品提供宿主驱动和 API 。用户届时将可以独立使用 UHD 驱动而无需 GNU Radio。
​ USRP的硬件驱动器™（UHD）软件API支持在所有USRP SDR产品应用开发。因为它增加了代码的可移植性，允许应用程序无缝地过渡到其他USRP SDR平台开发时，要求扩大或新的平台上都可以使用一个通用的软件界面是非常关键的。因此，它使一个显著减少开发工作量，允许你保存和重用旧的代码，这样您可以专注于新的算法。
